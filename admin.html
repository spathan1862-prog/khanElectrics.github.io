JSON Format Guide

Your JSON file should be an array of objects like this:

[ { "Product Name": "LED Bulb 9W", "Category": "Lighting", "Price": "299", "Details": "Energy-saving LED bulb, 9W, warm white.", "Image URL": "https://example.com/led-bulb.jpg", "Status": "Available" }, { "name": "MCB Switch 32A", "category": "Wiring", "price": 450, "details": "32 Ampere MCB circuit breaker.", "image": "https://example.com/mcb.jpg", "status": "Available" } ]

Duplicate detection: Products with the same name (case-insensitive) are automatically skipped.
